# PanDaFiRe

Script Python pour :
- analyser des PDF
- déterminer le type de document
- identifier l'émetteur
- extraire la date
- renommer automatiquement les fichiers

## Fonctionnalités

- pdfplumber (PDF texte)
- OCR (Tesseract)
- classification par scoring (regex)
- logs JSON
